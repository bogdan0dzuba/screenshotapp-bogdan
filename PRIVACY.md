# Приватность ScreenshotApp Bogdan

ScreenshotApp Bogdan работает локально и не требует аккаунта.

- Изображения, проекты редактирования и OCR-текст сохраняются только в выбранной пользователем папке на Mac.
- OCR выполняется системным фреймворком Apple Vision на устройстве.
- Перед захватом локально сохраняются название приложения и доступный заголовок активного окна. Они нужны только для подписи строки истории, лежат в `*.project.json` рядом со снимком и никуда не отправляются.
- Облачной загрузки, аналитики, рекламы и телеметрии нет.
- Разрешение «Запись экрана» используется только после команды пользователя на создание обычного или прокручиваемого снимка.
- Пункт «Проверить обновления…» открывает страницу последнего GitHub Release в браузере; фоновых сетевых запросов приложение не выполняет.
- Очистка истории затрагивает только файлы ScreenshotApp с ожидаемыми именами и перемещает их в Корзину.
